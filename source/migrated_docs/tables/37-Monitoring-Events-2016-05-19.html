Event Description
CREATED A new monitor was created
CANCELLED The monitor was deleted
ACTIVATED The monitor was reactivated
DEACTIVATED The monitor was deactivated
RESULT_CHANGE The result of the monitor check was different this time than the last.
STATUS_CHANGED The monitored system status is different than that reflected by the test results. This can fire twice on execution systems, once for each test.
PASSED A check passed
FAILED A check failed
UNKNOWN A check resulted in an unknown result
